# doggy_shelter

To run the demo repo code:
1. Fork and clone this repo
2. cd into repo
3. Install requirements
   - `pipenv install`
4. Activate your virtual environment
   - `pipenv shell`
4. cd into project and start server
   - `python manage.py runserver` 
5. See the app in action at
   - `http://localhost:8000/`
